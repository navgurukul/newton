### `Boolean` data type

`Boolean` is a common data type of all `programming` languages. `Boolean` returns two values.

*`True`*

*`False`*

**Example :-**

```python
a=("")
print(bool(a))
```
**Output :-**

`False`

`Because empty string is false.`

**Example :-**
```python
b=("megha")
print(bool(b))
```
**Output :-**
`True`

`Because any string is true except empty.`

**Example :-**
```python
print(True)
print(type(True))
print(False)
print(type(False))
```
**Output :-**

`True`

`<class bool>`

`False`

`<class bool>`
