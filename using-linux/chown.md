## chown

```bash
touch somefile.txt # `somefile.txt` naam ki file create karo
ls -l somefile.txt # `somefile.txt` file ki permissions dekho
chown root:root somefile.txt # `somefile.txt` file ki permissions root ko de do
ls -l somefile.txt
```
