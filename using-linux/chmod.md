## chmod

```bash
touch random.txt            # `random.txt` naam ki file create karo
ls -l random.txt            # `random.txt` file ki permissions dekho

chmod 755 random.txt
ls -l random.txt

chmod g+w random.txt
ls -l random.txt

chmod o+x random.txt
ls -l random.txt

chmod g-r random.txt
ls -l random.txt
```
